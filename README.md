### [gauravbisht.tech](https://gauravbisht.tech)

First, clone this Project:

```bash
git clone https://github.com/iamgauravbisht/portfolio.git
```

second, install dependencies:

```bash
npm install
# or
yarn
# or
pnpm
# or
bun
```

third, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
